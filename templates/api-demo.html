API Vulnerability Demonstrations

🔍 Уязвимые API эндпоинты:

  • GET /api/user/<id> - XSS через API
  • GET /api/users - IDOR (доступ к данным всех пользователей)
  • GET /api/comments - Хранимый XSS в комментариях
  • GET /api/admin/stats - Утечка информации (без проверки прав)
  • GET /api/search?q= - SQL инъекция
  • GET /api/export - Экспорт всех данных

Тестирование API:

Примеры использования:

// XSS через API
fetch('/api/user/1').then(r=> r.json()).then(d => console.log(d));

// SQL инъекция
fetch('/api/search?q=' OR '1'='1').then(r=> r.json()).then(d => console.log(d));

// Получение всех данных
fetch('/api/export').then(r=> r.json()).then(d => console.log(d));
